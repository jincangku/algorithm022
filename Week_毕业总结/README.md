学习笔记

2020年12月，我通过微信公众号了解到了极客大学的算法训练营的课程，以及硅谷大佬超哥。我毫不犹豫地就报了名并参加了课程。在参加这个课程之前，算法的困难一直萦绕着我身边，让我感到无从下手。自从上了这个课之后，我感觉算法一下子变得前所未有的条理清晰和简单。

超哥教给我最大的一个诀窍是——五毒神掌。以前我做做算法题的时候，从来都只是做一遍，也没有复习和重做的机制。超哥的“五毒神掌”确实是个学算法的好办法，一道题目做五遍才能真正明白它的内涵和道理。除此以外，极客大学提供的交流和学习平台也让我大大的拓宽了眼界，了解到了算法更深层次的一些细节。

LeetCode国际版也让我收获良多。在英语的平台上我可以看到全世界的算法精英互相交流和切磋，并从他们的好的点子中收获精华。

我希望未来极客大学能开设更多这样的算法和计算机的课程，我也期待着和更多优秀的老师和同学一起学习和进步。
